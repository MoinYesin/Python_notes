Basic Concepts

2. Conditionals

Conditionals are expressions that evaluate to either true or false.

if else:

Ex: 

                if price < 500 :
                    print("i will buy it")
                else :
                    print("i won't buy it")
In the example above: if the price is less than 500, then print "i will buy it". if the price is not less than 500, then print "i won't buy it". 
                if price < 500 :               -- if this condition is true
                    print("i will buy it")     -- then do this.
                else :                         -- if above condition is false
                    print("i won't but it")    -- then do this.
**if else can be used to make conditions and making decisions.

Code Indentation

If something is under something else, we need to indant that.
Ex: The allignment of the replies of a comment on a Facebook post. The replies goes a little bit to the right side of the actual comment. So, we can notice that it's a reply to that comment.(Illustrated below)
Example code: 

              price = 7
              if price < 10:
                  print("i want that food")
In the above code we want to execute the "print" command when the "if" condition is true. We have to indent the third line. That's how Python understands to execute.
**If you don't indent the line, you will get an Indentation error.
**If you have multiple lines inside the "if" block, all the lines need to be indented and the indentation has to be same.
**Anything indented is called a code block. Like the "if" block on the above example.

Equal or Not

1) To compare something smaller or less than, we use "less than"(<) sign.
2) To compare something bigger or greater than, we use " greater than" (>) sign.
3) To compare exact equal, we use two "equal" (==) sign.
**A single equal(=) is used to assign a value to a variable. Don't get confused.
4) To compare not equal, we use "not equal" (!=) sign.

Adding more conditions

1) To add more conditions to an "if else" statement, we need to add a "and" keyword in between the conditions in the first line.
Example Code: 

    salary = 5700
    vackation days = 23

    if salary > 5000 and vackation days > 40 :
        print("I will take this job")
    else :
        print("This job sucks! Give it to him")
2) To combine conditions we can use "or" . In this case only one condition has to be true in an if statement.
Ex: 
    if time == 5AM or time == 6AM:
        print("GOOD MORNING")

elif :

The "elif" condition is used to include multiple conditional expressions after the condition or between the "if" and "else" condition.
Example code: 

    price = 100

    if price > 100 :
        print("price is greater than 100")
    elif price == 100 :
        print("price is 100")
    elif price < 100 :
        print("price is less than 100")

Output:- price is 100
