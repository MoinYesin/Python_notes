Basic Concepts

1. Variables

Variable is something that could vary.(Vary+able)
Ex: Your age, It changes every year.

**Use a Variable to write something that could change.


*How to write variables?

All we need :
1. Variable name
2. An equal sign(=)
3. The value
Thats it.
Example code: 

    Likes = 43
Here, "Likes" is Variable name, then Equal sign (=), and then the Value (43).

*Variable Types:

1. String: When a value of a Variable contains text (one or more words), it is called s String.
Ex: 

 name= "Moin Yesin"

*While writing a string type variable, the volume of the variable has to be inside double quotes(".."). Like the example above.(We can also use single quotes('..')).

2. Number: When the value of a Variable is number, it's called a numeric type Variable.
Ex: 
 age= 15

Number Variable Types:
a) int:- There are few types of numbers. If a number does'nt have a fraction, it is called intager number. 15,31, 123 etc.
b) float:- When a number has a fraction, they are called float numbers. 2.5, 13.25 etc.

** Use int or float to convert a number inside quotes to a number.
** We will discuss more about int and float in the chapter Input

3. Boolean: A Variable whose value could be either "True" or "False" is called a Boolean type Variable.
Ex: 
isRich = True
isScared = False

**True or False must start with uppercase(only first letter) and does'nt need double quotes.
