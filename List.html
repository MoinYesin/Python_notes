Fundamentals

List

We can add more than one value in a variable. When we put more than one value in a variable, it is called a "List", or "Collection" or "Array"
Each item in a list is called an element of the list.
Example code: 

                friends_age = [17,19,21,15,16]
**While declaring a list or array you need to do two things-
1) Put multiple values inside square brackets "[...]"
2) Use a comma "," between elements.
**To create a list with strings, we have to enclose each name with quotation marks.
Example code: 
                friends = ["Moin", "Harish", "Anshu"]

Index:-

Each element in a list has a position. This position is called an "Index". The index of the first element in a list starts with 0 and then increases by 1.
Example code: 

                heroes = ["batman", "catman", "ironman"]
Here the index of batman is 0, The index of catman is 1 and index of ironman is 2.
We can get the value of an element by its index like this-
Example code: 
                heroes = ["batman", "catman", "ironman"]
                print(heroes[2])

            Output:- ironman

Get the index:-

If you are not sure what is the index of an element in a list, you can find out like this- 

                heroes = ["batman", "catman", "ironman"]
                catman_index = heroes.index("catman")
                print(catman_index)
            Output:- 1
Example2:- 
                heroes = ["batman", "catman", "ironman"]
                print(heroes.index("catman"))
            Output:- 1

Set by index:-

You can set or replace the value of an element by index.
Example code:- 

                heroes = ["batman", "catman", "ironman"]
                heroes[1] = "spiderman"
                print(heroes)
            
            Output:- ["batman", "spiderman", "ironman"]
The above code replaced "catman" with "spiderman".
