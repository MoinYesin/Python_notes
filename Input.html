3. Input

To ask for user input in Python type "input()". This will ask the user to provide some information. 

    input()
While asking for user input, you can add extra info to tell the user what information they should provide.
Example code: 
    input("Tell me your favorite food: ")
You can also store user input in a variable and then use that variabkle.
Ex: 
      name = input("Who is your bestfriend: ")
      print("Best friend: " + name)
**By-default user input is taken as a String. No matter if user inserts a text or a number, both are taken as a String.
If you expect a number, use special keyword "int" or "float" to convert user input to a number.
Example code: 
      siblings_text = input("Number of siblings: ")
      siblings = int(siblings_text)
      print(siblings)
Important
** Let's say you are asking for user's age. In this case, the user will provide an integer number. You will use "int" to convert the input to a number.
And if you are asking for currency rate or price, You will use "float". Because user input might contain a fraction.
**In some cases, you might not sure whether a user will insert an integer ot a float number. In these cases, use "float" to be on the safe side.
